Responsive Test Instructions

Test the following screen sizes:

  • Mobile (320px - 767px):
    • Navigation should show hamburger menu
    • Cards should stack in single column
    • Buttons should be full width
    • Text should scale appropriately

    • Buttons should be inline

    • All elements should show properly
    • Cards should show 3 columns
