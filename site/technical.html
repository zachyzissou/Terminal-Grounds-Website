Technical Details

This site is a static build served by Nginx in Docker. CI publishes images to GitHub Container Registry. Use the health endpoint for monitoring.

Stack

FrontendHTML5, CSS3, JS (ES6)
ServerNginx (Alpine)
ContainerDocker + Compose
RegistryGHCR

Endpoints

Health: GET /health → 200 OK

Assets cached 1y, HTML no-cache.

Run Locally

Python: cd site && python -m http.server 8000

Docker: docker compose up -d → http://localhost:2161

Security

Security headers, optional HSTS behind TLS, hidden files denied.
